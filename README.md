# Generate Combinations

## Installation
```
git clone https://github.com/mhertovm/GenerateCombinations.git
cd GenerateCombinations
npm instal
```
## Database
 create .env file
 ```
PORT = your_app_port
DB_HOST = your_host
DB_USER = your_username
DB_PASSWORD = your_password
DB_NAME = your_db_name
```
## Run
```
npm start
npm run dev
```
